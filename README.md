# UnityPractice_04
Udemy course practice project.

這是一款塔防遊戲，透過建設防禦塔，阻止敵人抵達我方城堡。

滑鼠點綠地可以花費黃金建設塔，敵人會越來越強。

透過將地板網格化，製作巡路邏輯。
